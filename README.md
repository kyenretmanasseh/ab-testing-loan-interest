# ab-testing-loan-interest
loan interest rates using A/B testing and python
## Project Overview
This project stimulates an A/B test to compare two loan strategies used by a financial institution.

Group A and Group B are analyzed based on:
- Conversion rate
- Total  profit
- Average Loan Size

  Tools Used
  - Python
  - Numpy
  - Pandas
  - Matplotlib
  - Statsmodels
 
  ## Key Findings
  Group B achieved a higher conversion rate and generated higher total profit compared to Group A.
  The experiment shows that the strategy used in Group B attracts more borrowers and produces greater overall profitability.
